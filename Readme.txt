Модуль обмена для решения "Агент Плюс: Мобильная торговля (базовая)" и конфигураций на базе платформы 1С:Предприятие 8.2
версия 2.0.5.19
=================================================================

В архиве находятся файлы внешних обработок для 1С:Предприятия:

1. апОбменДанными10.epf - основная обработка для проведения настроек и обмена данными.
2. апЖурналРегистрации.erf - служебный отчет, формирует журнал обмена данными. 
3. апЗагруженныеДокументы.erf - служебный отчет, формирует список загруженных документов от агентов. 

Модуль совместим со следующими конфигурациями:

 * «1C:Управление торговлей», версии 10.3; 
 * «1C:Управление торговлей алкогольной продукцией», версии 10.3;
 * «1C:Комплексная автоматизация», версии 1.1;
 * «1С:Управление производственным предприятием», версии 1.3;
 * «1С:Управление торговым предприятием для Казахстана», версии 2.0;
 * «1С:Управление торговлей для Казахстана», версии 2.2.

Внимание!
Текущая версия модуля предназначена для совместного использования с типовой конфигурацией "Управление торговлей, версии 10.3" не ниже 10.3.8.9!
 
Установка и запуск модуля
-----------------------------------------------------------------
1. Распаковать архив в любое место на диске.
2. Запустить основную обработку "апОбменДанными10.epf".  
3. Выполнить настройку обмена данными и агентов.
