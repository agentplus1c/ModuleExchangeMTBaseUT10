Документ «Планирование для агентов»

Документ используется для составления планов посещений торговых точек или клиентов для торговых агентов. Представляет собой совокупность типовых документов "Событие", у которых реквизит "Группа события" имеет предопределённое значение "Маршрут" (один документ "Событие" - одна точка маршрута). Внимание: изменение вручную документов "Событие", используемых документами "Планирование для агентов", может изменить список точек маршрута в существующих планах посещений!

Планирования посещений:

Предусмотрено два типа формирования документа (в зависимости от того, установлен флажок "Использовать торговые точки" или нет):

  • Использование плана посещений с использованием торговых точек. В этом случае в табличной части документа будут отображаться колонки "Торговая точка" и "Контрагент".
  • Использование плана посещений с использованием клиентов. В этом случае в табличной части документа будет отображаться только колонка "Контрагент".

При формировании документа "Планирование для агентов" необходимо выбрать агента, для которого формируется данный план, а также интервал планирования, который может быть равен месяцу, недели, 2 неделям, 4 неделям или быть заданным пользователем.

Если установлен флажок "Использовать время", то для точки маршрута становится доступно для ввода поле "Время"; если установлен флажок "Использовать комментарий" - становится доступно для ввода поле "Комментарий".

Установив флажок "Использовать порядок", можно указать порядок посещения точек маршрута.

Для удобства заполнения точек маршрута документа могут применяться кнопки, расположенные на командной панели табличной части:

  • Кнопка "Заполнить" производит заполнение текущего плана посещений на основании другого плана.
  • Кнопка "Подбор" используется для подбора клиентов/торговых точек.
  • Кнопка "Сортировка" предназначена для сортировки введенных значений по клиентам.

Для удобства редактирования точек маршрута на форме предусмотрена возможность их редактирования с помощью контекстного меню (для этого нужно кликнуть правой кнопкой мыши по нужной ячейке в текущей строке табличной части и выбрать необходимое действие), либо с помощью кнопок, расположенных в командной панели табличной части "План посещений".
