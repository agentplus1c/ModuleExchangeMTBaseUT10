Отчет "Выполнение плана агентами"

Отчет по выполнению плана посещений строится на основании документа "Планирование для агентов" и документов, полученных из МУ.

Отчет может выводиться за произвольный период и включать в себя несколько документов "Планирование для агентов".

Кнопка «Отбор» на панели управления предназначена для отображения/скрытия панели, на которой выбираются параметры формирования отчета.

Планирование посещений агентами

С помощью значений отбора, представленных на форме отчета, можно отобрать документы по следующим критериям: по типам документов или по агенту. Также с помощью установки соответствующего флага можно отобрать только проведенные документы.

Печатная форма отчета по плану посещений выполнена в стиле печатной формы документа "План посещений", отражает три вида состояний: "не посещено", "посещено по плану" и "посещено вне плана".

Колонка отчета «Торговая точка» может отображаться или не отображаться в зависимости от того, установлена ли константа "Использовать торговые точки" в настройках параметра обмена.

При двойном клике на ячейке отчета с символами "Посещено по плану" или "Посещено вне плана", на экране отображается контекстное меню со списком документов, оформленных в соответствующий день для контрагента в текущей строке отчета. При нажатии на соответствующий документ в контекстном меню он открывается в отдельном окне.

Желтым цветом выделяются те строки, по которым не было запланировано посещение, но были сформированы какие-либо документы агентом в отчетном периоде.
